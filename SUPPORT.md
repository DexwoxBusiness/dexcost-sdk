# Support

Need help with DexCost SDKs or the DexCost cloud? Use the channel that best fits
the request.

| Need | Contact |
| --- | --- |
| SDK integration help, product questions, bugs | support@dexcost.io |
| Security vulnerabilities | security@dexcost.io |
| Code of Conduct concerns | conduct@dexcost.io |
| Billing, invoices, refunds, GST documents | billing@dexcost.io |
| Sales, enterprise plans, partnerships | sales@dexcost.io |
| Legal, privacy, contracts, notices | legal@dexcost.io |
| General questions | hello@dexcost.io |

For open-source bugs and feature requests, GitHub issues are welcome when the
topic is not security-sensitive. Please include the SDK language, package
version, runtime version, and a minimal reproduction where possible.

Never post API keys, customer data, production traces, or security vulnerability
details in a public issue.
